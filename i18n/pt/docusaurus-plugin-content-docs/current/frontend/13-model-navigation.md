# 13. Navegação de Modelos e Selecção de Cenários

## 13.1 Alternar Entre Modelos de Dados

A barra lateral mais à esquerda lista todos os Modelos de Dados públicos como botões de ícone, ordenados pelo campo **Ordem de Apresentação**. Clique em qualquer ícone para activar o modelo — os três painéis actualizam. Um menu de alternância de modelos também está disponível no cabeçalho do painel de controlo para troca rápida.

## 13.2 Seleccionar um Cenário

O menu suspenso de cenários perto do topo do Painel de Filtros lista todos os cenários disponíveis para o modelo activo, ordenados pela **Ordem de Apresentação**. Apenas os cenários com um FicheiroCenário em estado **Pronto** aparecem.

O cenário seleccionado é codificado no URL (`?scenario=...`) para marcação e partilha.
