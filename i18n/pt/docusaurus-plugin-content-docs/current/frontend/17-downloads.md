# 17. Transferências

A página de Transferências (acessível a partir da navegação do cabeçalho) lista todos os Conjuntos de Dados Vectoriais, Conjuntos de Dados Raster e Conjuntos de Dados de Referência a que tem acesso. Os utilizadores não autenticados vêem apenas os conjuntos de dados públicos e aprovados no painel de camadas de cada modelo.

Cada cartão mostra o nome, descrição, atribuição de fonte e data da última actualização. Clique em **Transferir** para transferir o ficheiro fonte bruto no seu formato original.

Use a caixa de pesquisa no topo para filtrar conjuntos de dados por nome ou descrição.

:::note
As transferências de dados vectoriais reflectem o **ficheiro original carregado** (GeoJSON, GeoPackage, Shapefile ou KML), não os PMTiles processados. Carregue-os directamente no QGIS, ArcGIS ou ferramentas similares.
:::
