# 15. O Painel do Mapa

## 15.1 Controlos e Navegação do Mapa

| Controlo | Descrição |
|---|---|
| Zoom | Roda do rato, gesto de pinça ou os botões +/-. |
| Deslocamento | Clicar e arrastar. |
| Recentrar | Clique no botão de recentrar (canto inferior direito) para regressar à extensão predefinida de Moçambique. |
| **Geocodificador (pesquisa de endereço)** | Barra de pesquisa no canto superior esquerdo. Pesquise nomes de lugares via OpenStreetMap Nominatim. As localizações correspondentes são mostradas com um marcador. |
| **Botão de Partilha** | Copia o URL da página actual (incluindo todos os filtros activos e cenário) para a área de transferência para partilha. |

## 15.2 Barra de Ferramentas de Controlos do Mapa

| Controlo | Descrição |
|---|---|
| **Selector de mapa base** | Alterne entre estilos de mapa base claro, escuro e outros disponíveis. |
| **Controlo de opacidade** | Ajuste a opacidade da camada de dados do cenário activo. |
| **Legenda** | Chave de cores para a coluna de visualização activa. Corresponde à configuração `color_coding`. |
| **Painel de camadas contextuais** | Activar/desactivar camadas vectoriais individuais (das Camadas Contextuais do Modelo de Dados). |
| **Painel de camadas raster** | Activar/desactivar camadas raster individuais (das Camadas Raster do Modelo de Dados) como sobreposições de mapa adicionais. |

## 15.3 Interacção com Características

Clicar numa característica no mapa muda o painel direito para o **modo de detalhe de característica**, mostrando os atributos configurados em `popup_fields`.

O cabeçalho do painel mostra o identificador da característica. Clique na **seta para trás** para regressar à vista de resumo nacional.
