# 9. Conjuntos de Dados de Referência e Ficheiros de Referência

Os Conjuntos de Dados de Referência armazenam documentos de suporte não espaciais e ficheiros tabulares — relatórios de metodologia, dicionários de dados, tabelas Excel, PDFs — que fornecem contexto para a análise de planeamento energético. Aparecem na secção de Transferências do frontend.

## 9.1 Adicionar um Conjunto de Dados de Referência

1. Vá a **Conjuntos de Dados → Conjuntos de Dados de Referência → + Adicionar Conjunto de Dados de Referência**.
2. Preencha os campos de metadados (nome, descrição, fonte, contacto, etc.). Suporta traduções EN/PT.
3. Clique em **Guardar**. Em seguida, adicione Ficheiros de Referência.

## 9.2 Adicionar Ficheiros de Referência

1. Vá a **Conjuntos de Dados → Ficheiros de Referência → + Adicionar Ficheiro de Referência**.
2. Seleccione o Conjunto de Dados de Referência pai.
3. Carregue um ficheiro. Formatos aceites: `.pdf`, `.csv`, `.xlsx`, `.xls`, `.docx`, `.doc`.
4. Clique em **Guardar**.

## 9.3 Ligar Conjuntos de Dados de Referência a Modelos de Dados

Adicione um Conjunto de Dados de Referência à multi-selecção **Conjuntos de Dados de Referência** de um Modelo de Dados para o associar ao contexto de análise desse modelo. Os documentos aparecem então na secção de Transferências, filtráveis por modelo.
